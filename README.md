# QA Advanced Git Lab 
 
Acest proiect demonstreaza gestionarea unui flux de lucru Git, incluzand refactorizarea, repararea bug-urilor si rezolvarea conflictelor de merge. 
 
## Ce am invatat: 
- Crearea si gestionarea branch-urilor paralele. 
- Identificarea si rezolvarea manuala a unui conflict de tip Merge. 
- Utilizarea constantelor in Java pentru o structura mai curata.
